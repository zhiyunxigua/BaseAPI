# 文字面板使用说明

## 一、适用场景

文字面板用于给指定玩家显示一个 3D 文本框，适合：

- NPC 头顶提示
- 地点说明牌
- 副本引导文字
- 临时任务提示
- 实体跟随标签

---

## 二、基础用法

### 1. 世界坐标面板

```java
import com.xigua.baseAPI.api.TextBoard;

TextBoard board = TextBoard.create("欢迎来到主城")
        .bindToWorld(100, 65, 200)
        .boardColor(0, 0, 0, 120)
        .textColor(255, 255, 255, 255)
        .faceCamera(true)
        .build();

board.send(player);
```

### 2. 绑定实体

```java
TextBoard board = TextBoard.create("任务管理员")
        .bindToEntity(npc, 0, 2.2, 0)
        .boardColor(0, 0, 0, 100)
        .textColor(255, 255, 0, 255)
        .build();

board.send(player);
```

---

## 三、修改面板

### 1. 修改文本

```java
board.setText("任务已完成").send(player);
```

### 2. 修改颜色

```java
board.setTextColor(0, 255, 0, 255).send(player);
board.setBoardColor(0, 0, 0, 160).send(player);
```

### 3. 修改位置或偏移

```java
board.setPos(120, 70, 220).send(player); // WORLD 模式
board.setOffset(0, 2.5, 0).send(player); // ENTITY 模式
```

### 4. 修改旋转和缩放

```java
board.setRot(0, 90, 0).send(player);
board.setScale(1.4, 1.4).send(player);
```

---

## 四、删除与重置

### 1. 删除单个面板

```java
board.delete(player);
```

### 2. 主动重置当前玩家的文字面板

```java
TextBoard.init(player);
```

适用场景：

- 你准备重新下发一整套场景面板前
- 你需要手动清空当前玩家已有面板时
- 客户端状态被你自己的业务主动重置时

典型流程：

```java
TextBoard.init(player);

TextBoard.create("矿区入口")
        .bindToWorld(20, 64, 30)
        .send(player);

TextBoard.create("铁匠铺")
        .bindToWorld(35, 64, 18)
        .send(player);
```

---

## 五、批量发送

```java
Collection<? extends Player> players = Bukkit.getOnlinePlayers();

TextBoard board = TextBoard.create("双倍活动进行中")
        .bindToWorld(0, 80, 0)
        .textColor(255, 215, 0, 255)
        .build();

board.send(players);
```

更新时同样可以批量发送：

```java
board.setText("活动已结束").send(players);
```

---

## 六、推荐实践

1. 长期存在的面板保留 `TextBoard` 实例，后续直接 `setXxx().send(...)`。
2. 玩家进入服务器后的首次 `init` 已由 BaseAPI 自动处理，业务侧只需要关注自己的创建和更新逻辑。
3. 如果绑定目标实体需要变化，不要硬改旧实例，直接新建一个面板更稳妥。
4. 颜色统一按 `0-255` 传入，客户端会自动转换。
5. 对不同玩家展示不同内容时，复用同一个实例前要明确该实例的状态是否还适合所有玩家。

---

## 七、BaseAPI 便捷写法

如果你已经持有 `BaseAPI` 插件实例，也可以这样写：

```java
baseAPI.createTextBoard("副本入口")
        .bindToWorld(10, 65, 10)
        .send(player);
```

---

## 八、限制说明

1. `setPos` 仅适用于 `WORLD` 面板。
2. `setOffset` 仅适用于 `ENTITY` 面板。
3. 当前不支持直接修改绑定实体 ID，如需切换目标实体，请删除旧面板并创建新面板。
